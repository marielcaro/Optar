What is this?

The Xojo IDE allows you to change the color of various items in the code editor. Keywords, strings, integers etc can all have different colors, to aid readability or simply for personal preference. XojoThemes allows you to change these colors in groups (or themes).


Why?

If I’m working in Xojo during the day, I often use the Solarized Dark theme but at night I use the Dusk, which is easier on my eyes. I can choose these colors one by one in the Xojo Preferences window but it’s a little slow to do this often. XojoThemes allows you to change themes quickly.

What does it actually do?

It uses a Xojo Shell to write to the macOS defaults. On Window, it writes to the Registry. On a Mac, you can see these for yourself by opening up a Terminal and typing defaults read com.xojo.xojo. On Windows, regedit.exe is your friend.


How to use it

Select a theme. You’ll see a preview. Like it? Then click the Set Theme button - no changes are made before this. Relaunch the Xojo IDE if it's open and bask in the glory of your chosen theme.

Make your own theme

Select a pre-existing theme to base your new theme on. Then click the Add button, below the list of themes. Double-click the new theme to change its name. Any changes you make to the theme are automatically saved to XojoThemes.


Import themes

XojoThemes opens XML documents with the extension .xojo_theme. Go to the File menu and select Import or right-click on the theme. Or, simply drag the theme into XojoTheme's theme list.


Export themes

To export a theme, go to the File menu and select Export. Or simply right-click on the theme in the theme list. Remember to send me your favorites so I can share them on the XojoThemes website.


Requirements

XojoThemes is currently Windows and macOS only. It is generally tested with the last couple of releases of Xojo, in this case the Xojo 2016 releases. It should, however, work just fine with previous versions. YMMV.
